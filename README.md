# Clicky Cards

React App.

### How to play?

- There are 5 levels in the game.
- Each new level add 2 more cards.
- You have to click every card but try not to hit any duplicates!


### Built with

- JavaScript
- React.js
- Node.js
- CSS
- SVG
- Github for versioning
